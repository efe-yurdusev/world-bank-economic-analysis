# 📊 World Bank Economic Analysis

This project analyzes the relationship between **GDP per capita** and **population** using data from the World Bank API.

---

## 🔧 Technologies Used
- Python  
- Pandas  
- Matplotlib  
- World Bank API  

---

## 📁 Dataset
- Source: World Bank API  
- Indicators:
  - GDP per capita (current US$)
  - Total population  

---

## 📊 Analysis

### 1. Population vs GDP per Capita
- Scatter plot used to observe relationship  
- Bubble size represents population  
- Log scale applied  

### 2. GDP per Capita Over Time
- Line chart used to compare countries  
- Shows long-term economic trends  

---

## 📌 Key Insights
- Germany has stable economic growth  
- Japan’s growth slowed after a period  
- Turkey shows growth but with fluctuations  
- Population growth does not always mean economic growth
